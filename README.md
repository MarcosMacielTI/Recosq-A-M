# Reconquistar Adri

Um aplicativo React romântico interativo criado com Vite e Tailwind CSS.

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra o navegador em `http://localhost:5173`

## Personalização

- Substitua as imagens no array `memories` em `src/App.jsx` pelas suas fotos.
- Atualize o link do WhatsApp com o seu número.

## Tecnologias

- React
- Vite
- Tailwind CSS
- Lucide React (ícones)
